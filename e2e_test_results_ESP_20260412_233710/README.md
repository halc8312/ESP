## 主要なリスク概要
1. **編集系画面の外部CDN依存**: TinyMCE/Sortable/Chart.js をCDN直読みし、一部画面はフェイルセーフがなくJSエラー化する。
2. **外部API/外部画像依存**: 公開カタログは為替API不達時にフォールバックするが、外部画像/フォント/CDNへの到達性に表示品質が依存する。
3. **未実施領域**: CSV入出力、スクレイピング失敗制御、アーカイブ/ゴミ箱、モバイル表示、複数タブ/複数ログインの整合は残件。
4. **公開導線の運用依存**: カタログ公開URLは期待どおり機能したが、本番のCDN/CSP/ネットワーク条件で編集UXが崩れる余地がある。

## 置いた仮定
- 実環境URL未提示のため、ローカル起動環境を対象環境として合理的に代替した。
- ロール差分仕様が未提示のため、ユーザー間分離を主要な権限制御確認対象とした。
- 外部送信や実サイトスクレイピングは安全性と環境制約を優先し、必要時はBlocked/Not Runとして記録した。
